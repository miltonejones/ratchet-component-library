# Stack

The Stack component manages layout of immediate children along the vertical or horizontal
axis with optional spacing and/or dividers between each child.

<Canvas>
  <Story name="Stack" parameters={{ layout: "fullscreen" }}>
    <Stack spacing={2}>
      <Box>Item 1</Box>
      <Box>Item 2</Box>
      <Box>Item 3</Box>
    </Stack>
  </Story>
</Canvas>

```jsx
<Stack spacing={2}>
  <Box>Item 1</Box>
  <Box>Item 2</Box>
  <Box>Item 3</Box>
</Stack>
```

# Flex

Flex is Box with display: flex and comes with helpful style shorthand. It renders a div element.

## Using the Spacer

As an alternative to Stack, you can combine Flex and Spacer to create stackable and responsive layouts.

<Canvas>
  <Story name="Flex" parameters={{ layout: "fullscreen" }}>
    <Flex>
      <Box p="4" bg="red">
        Box 1
      </Box>
      <Spacer />
      <Box p="4" bg="green">
        Box 2
      </Box>
    </Flex>
  </Story>
</Canvas>

```jsx
<Flex>
  <Box p="4" bg="red">

  </Box>
```

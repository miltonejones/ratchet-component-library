# Menu

Menus display a list of choices on temporary surfaces.

## Basic menu

A basic menu opens over the anchor element by default (this option can be changed via props).
When close to a screen edge, a basic menu vertically realigns to make sure that all menu items are completely visible.

Choosing an option should immediately ideally commit the option and close the menu.

<Canvas>
  <Story name="Menu" parameters={{ layout: "fullscreen" }}>
    {MenuTemplate.bind({})}
  </Story>
</Canvas>

```jsx
<Menu
  onChange={(e) => alert(options[e])}
  button={<Button>dashboard</Button>}
  options={options.map((b) => (
    <Box p={1} key={b}>
      {b}
    </Box>
  ))}
/>
```
